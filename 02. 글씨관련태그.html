그 밖에 글자를 다루는 태그들

일반글꼴(태그로 감싸지 않은 텍스트)

b: 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더(읽을때 강조)

i : 글자를 기울여서 표시하는 태그

em(emphasis) : 기울여서 표시하는 태그 + 스크린리더(읽을때 특히 강조해서)

mark : 형광펜 의 효과를 주는 태그

u : 글자에 밑줄을 그어주는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현하는 태그

기본글자에 sub : 아래첨자 를 나타내는 태그
sup : 위첨자 를 나타내는 태그

abbr : 줄임말 표시
