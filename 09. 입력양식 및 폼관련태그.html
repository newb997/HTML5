form태그내에 submit버튼 클릭시 해당 form안에
사용자가 작성한 입력한 값들을 서버로 전송하면서 요청하는 역할 수행

form action="" method=""

* action속성 : 해당 form에 사용자가 입력한 값을 전달하면서 요청할 서버의 경로제시 * method속성 : 요청 전송 방식을 지정하는 속성(get(기본값) | post)
> get 방식 : 서버에 요청시 사용자가 입력한 값들이 url에 노출되는 방식
> post 방식 : 서버에 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form내에 submit버튼 클릭시 action이 지정한 서버로 사용자가 입력한 값들이 넘어가면서 요청됨(사용자가 넣은값은 key = value)

라디오 버튼과 체크박스 관련 태그

라디오 버튼(type="radio")

- 여러 값들 중 한개만 선택가능 - 서버로 value에 들어있는 값이 전송됨 - name속성의 값이 같아야 한개만 선택할 수 있음

성별 : 선택안함   남자   여자

성별2 : 선택안함   남자   여자



체크박스(type="checkbox")

- 제시한 여러개의 값들 중 여러개를 선택할 수 있음 - 서버로 value의 값이 전송됨
